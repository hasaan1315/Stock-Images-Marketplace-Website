<h1>LensLegacy - Painting Stories with Pixels</h1>

<h2>Description</h2>
LensLegacy is an online marketplace crafted for photographers and visual artists to showcase and sell their high-quality images. It transforms the visual content landscape by enabling creators to upload images, set prices, and define licensing terms. Buyers can explore diverse categories and securely purchase images with flexible usage rights. The platform ensures secure user authentication, seamless transactions, and a supportive environment for collaboration and creativity.

<h2>Languages and Technologies Used</h2>
- <b>HTML</b>  
- <b>CSS</b>  
- <b>JavaScript</b>  
- <b>Firebase</b>  

<h2>Environments Used</h2>
- <b>Web Browsers</b> (Chrome, Firefox)
- <b>Firebase Console</b>
- <b>Code Editor:</b> Visual Studio Code

<h2>Key Features</h2>

- 🔐 <b>User Authentication:</b> Secure login, registration, and session management.  
- 🖼️ <b>Upload & Showcase:</b> Artists can upload photos with tags, pricing, and licensing.  
- 🔎 <b>Browse & Search:</b> Intuitive navigation through categories like Nature, Travel, Food, and Tech.  
- 💳 <b>Purchase & License:</b> Flexible licensing options with secure payment handling.  
- 📦 <b>Transaction Management:</b> Firebase Firestore and Authentication handle all backend logic securely.  
- 🧑‍🤝‍🧑 <b>Community-Driven:</b> LensLegacy encourages collaboration, visibility, and artistic exploration.
